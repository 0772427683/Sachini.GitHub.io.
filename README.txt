# Sachini Bandara Portfolio

This is the source code for my personal portfolio website. It showcases my skills, projects, education, and more as a biotechnology student at SLIIT.

## Features:
- **About Me**: Introduction and bio.
- **Education**: Information about my educational background.
- **Skills**: Display of various skills using progress bars.
- **Projects**: Overview of my recent projects.
- **Contact**: Links to my social media and contact information.

## Technologies Used:
- HTML
- CSS
- Font Awesome for icons

## How to Use:
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the portfolio.
3. Modify the `styles.css` file to customize styles.
